
# 💰 Expense Tracker

An easy-to-use Expense Tracker web application built using **HTML, CSS, and JavaScript**. It helps users keep track of their income and expenses, calculate their current balance, and manage transactions through a clean and responsive interface.

## 🚀 Features

- 🏠 Welcome Page
- ➕ Add income and expense transactions
- 🗑️ Delete transactions
- 💵 Automatically calculate:
  - Total Balance
  - Total Income
  - Total Expenses
- 📱 Responsive design
- 🎨 Simple and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
